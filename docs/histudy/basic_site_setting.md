## One page Navigation

1. Prepare the **page**
2. Create a new page. To do that go to Pages Add New  and give the page a title and then add a Block element to the content. Add section unique  **"ID"** into Per Block.
3. Prepare the **menu**
4. Go to Appearance Menu click the create a new menu link  Give the new menu a unique name and then click the Create Menu button.
5. Add menu item with **section ID**
6. Select Onepage menu for this page from page attributes options

## Turn on Coming Soon/Maintenance Mode

WordPress Maintenance Mode: What Is It and How to Use It. <https://elementor.com/blog/wordpress-maintenance-mode/>

1. Get back to the WordPress Dashboard and navigate to Elementor Tools Maintenance Mode
2. Choose between **Coming Soon** and **Maintenance Mode**
3. Pick the template for the **Coming Soon** Mode. Template name **"Coming Soon"**
> **Note:** You will see a red button on your top WordPress bar with the text: **" Maintenance Mode ON"**.
> **Tip:** You can use the same process described here for "Maintenance Mode"

![coming soon 1](img/histudy/coming-soon/1.png)

*Elementor > Tools > Maintenance Mode*

![coming soon 2](img/histudy/coming-soon/2.png)

*Choose between **Coming Soon** and **Maintenance Mode***

![coming soon 3](img/histudy/coming-soon/3.png)

*Pick the template for the **Coming Soon** Mode. Template name **"Coming Soon"** and click the save change button*

*Maintenance Mode ON*
