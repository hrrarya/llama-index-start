## Requirements

There are system requirements in order to install and setup Histudy theme and its components properly. Make sure that you are running the **latest version of WordPress,** **PHP version 5.6** or higher and **MySQL version 5.6** or higher. We also recommend the following PHP configuration limits. <https://wordpress.org/about/requirements/>

### Recommended PHP configuration limits

  
    max_execution_time = 300;
    upload_max_filesize = 128M;
    post_max_size = 128M;
    max_input_time = 300;
    memory_limit = 256M;
