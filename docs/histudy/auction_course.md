## Add Course[](#auction_course_title)

Please follow the instructions below:

1. From the dashboard go to Tutor LMS  Courses Add new and Name of the course
2. Course Description
3. Select Course Image
4. Select Course categories
5. Select Course tag
6. Course Short Discription
7. Course Additional Option
8. Tutor Course Meta
9. Author Select
10. Course Settings -> General
11. Add Product
12. Course Builder
13. Additional Data
14. Video
15. Click Publish Button
