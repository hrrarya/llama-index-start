## General

To change Histudy General Options navigate to Appearance Histudy Theme Options General

General menu contains such tabs as website logo settings.

The logo can be uploaded in 3 variants. This is very helpful in case you have pages with different backgrounds and are using a transparent header. It makes you available to choose the logo which looks the best for each of your pages individually.
