### **General Settings**

The settings are straightforward, giving you simple and direct control over which font formats are permitted.

- **Allow WOFF Font Uploads** Enabling this toggle allows you to upload font files in the **Web Open Font Format** (.woff). WOFF is a standard font format created specifically for use on the web, supported by all modern browsers.


**Allow WOFF2 Font Uploads** Enabling this toggle allows you to upload font files in the **Web Open Font Format 2.0** (.woff2). This is a newer format that offers significantly better compression than WOFF, resulting in smaller file sizes and faster website loading times. We highly recommend enabling this for optimal performance.
